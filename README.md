ATS Resume Analyzer:-

An AI-powered ATS (Applicant Tracking System) resume analyzer built using Flask and Google Gemini AI.
It evaluates resumes and provides an ATS score, strengths, weaknesses, and improvement suggestions.

Features:-
ATS score out of 100
Resume pros and cons
AI-powered suggestions
Clean Tailwind CSS UI
Tech Stack
Backend: Python, Flask, PyPDF2, Gemini AI
Frontend: HTML, Tailwind CSS, JavaScript

Setup:-

   Set Gemini API key:set GEMINI_API_KEY=your_api_key_here
   Install dependencies:pip install flask PyPDF2 google-generativeai
   Run the app:python app.py
